# Can-To-Uart
## 简述
这就是一个简单的小型Can逻辑分析仪，它可以接受CAN总线数据，并通过UART串口发送到上位机进行分析也能把数据通过OLED小屏幕显示出来。
## 产品优点
- 体积小巧，方便携带
- 低功耗设计，适合长时间使用
- 自带OLED显示屏，实时显示数据
- Key可以切换数据显示模式(10进制/16进制)
- 可自由选择是否接入120Ω终端电阻
- 可用Type-C接口供电，方便使用，不需要额外供电
- 集成了usb转ttl芯片，可直接通过数据线与电脑连接，无需额外的串口模块
## 开发环境

- Cubemx
